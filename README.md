## Чек-лист функциональных требований

### 1. Главная страница — Список объявлений (/list)
- [x] Отображение списка объявлений в виде карточек
  - [x] Изображение товара (placeholder)
  - [x] Название объявления
  - [x] Цена
  - [x] Категория
  - [x] Дата создания
  - [x] Статус (на модерации / одобрено / отклонено)
  - [x] Индикатор приоритета (обычный / срочный)
- [x] Фильтрация и поиск
  - [x] Фильтр по статусу (множественный выбор)
  - [x] Фильтр по категории
  - [x] Фильтр по диапазону цен
  - [x] Поиск по названию объявления
  - [x] Кнопка «Сброс всех фильтров»
- [x] Сортировка
  - [x] По дате создания (новые → старые / старые → новые)
  - [x] По цене (возрастание / убывание)
  - [x] По приоритету
- [ ] Пагинация
  - [ ] По 10 объявлений на страницу
  - [ ] Навигация между страницами (вперёд/назад, номера страниц)
  - [ ] Отображение общего количества объявлений

### 2. Страница детального просмотра объявления (/item/:id)
- [x] Переход по клику на карточку из списка
- [ ] Информация об объявлении
  - [ ] Галерея изображений (минимум 3, с переключением)
  - [ ] Полное описание
  - [ ] Характеристики товара (таблица ключ-значение)
  - [ ] Информация о продавце
    - [ ] Имя
    - [ ] Рейтинг
    - [ ] Количество объявлений
    - [ ] Дата регистрации
- [ ] История модерации
  - [ ] Список всех действий (по каждому — кто, когда, решение, комментарий)
- [ ] Панель действий модератора
  - [ ] Кнопка «Одобрить» (зелёная)
  - [ ] Кнопка «Отклонить» (красная)
  - [ ] Кнопка «Вернуть на доработку» (жёлтая)
  - [ ] При отклонении:
    - [ ] Обязательное поле для причины
    - [ ] Быстрые шаблоны причин:
      - [ ] Запрещённый товар
      - [ ] Неверная категория
      - [ ] Некорректное описание
      - [ ] Проблемы с фото
      - [ ] Подозрение на мошенничество
      - [ ] Другое (с полем ввода)
- [ ] Навигация
  - [ ] Кнопка «Назад к списку»
  - [ ] Кнопки «Предыдущее» / «Следующее» объявление

### 3. Страница статистики модератора (/stats)
- [ ] Общая статистика (карточки с метриками)
  - [ ] Всего проверено объявлений (за сегодня / неделю / месяц)
  - [ ] Процент одобренных
  - [ ] Процент отклонённых
  - [ ] Среднее время на проверку одного объявления
- [ ] Графики
  - [ ] Столбчатая диаграмма активности по дням (последние 7 дней)
  - [ ] Круговая диаграмма распределения решений (одобрено / отклонено / на доработку)
  - [ ] График по категориям проверенных объявлений
- [ ] Фильтр по периоду
  - [ ] Сегодня
  - [ ] Последние 7 дней
  - [ ] Последние 30 дней

## Чек-лист дополнительных возможностей

- [ ] Горячие клавиши
  - [ ] `A` — одобрить
  - [ ] `D` — отклонить

  - [ ] `←` — предыдущее объявление
  - [ ] `/` — фокус на поиск
- [ ] Bulk-операции
  - [ ] Чекбоксы для выбора нескольких объявлений

  - [ ] Счётчик выбранных объявлений
- [x] Продвинутая фильтрация
  - [x] Сохранение наборов фильтров
  - [x] Синхронизация фильтров в URL (возможность поделиться ссылкой)
- [x] Тёмная тема
  - [x] Переключатель темы
  - [x] Сохранение выбора в `localStorage`
- [ ] Экспорт данных
  - [ ] Экспорт статистики в CSV
  - [ ] Генерация PDF-отчёта
- [ ] Real-time обновления

  - [ ] Обновление статуса объявления без перезагрузки
- [ ] Анимации
  - [ ] Плавные переходы между страницами
  - [ ] Анимация появления карточек
  - [ ] Progress bar при загрузке данных
